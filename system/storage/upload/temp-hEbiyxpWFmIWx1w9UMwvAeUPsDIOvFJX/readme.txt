This is simple tool for skipping the Cart page on Opencart 2.3.x.x sites which are using Journal Theme. When customer click the "Add To Cart" button, system will redirect the customer to Checkout page...


Extract files
Just install it by using extension installer.
And don't forget to refresh the modification cache.
